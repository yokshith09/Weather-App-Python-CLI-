# Weather App (Python CLI)

A command-line Python application that fetches real-time weather data using the OpenWeatherMap API with a fallback dummy dataset.

## Features

- Live weather data using OpenWeatherMap API
- Dummy weather fallback for Indian cities
- Case-insensitive city search
- Displays temperature, humidity, wind speed, and weather condition
- Maintains last 5 searched cities

## Technologies

- Python 3
- Requests library
- OpenWeatherMap API

## Installation

Clone the repository:

git clone https://github.com/yourusername/weather-app.git

Navigate to the project folder:

cd weather-app

Install dependencies:

pip install -r requirements.txt

## Run the Application

python weather_data.py
